# Archive

This directory contains documentation and scripts from Phase 1 of the Menu Manager project.

These files are preserved for historical reference but are no longer actively maintained. The project is transitioning to a web-based submission system with multi-level approval workflows.

## Contents

### `/docs`
Historical documentation files including:
- Setup and configuration guides
- Testing documentation
- Implementation summaries
- Azure/Microsoft Graph integration notes
- Training guides
- Workflow documentation

### `/scripts`
Test and demo scripts from the original implementation:
- `test-menu.sh` - Menu testing script
- `test-workflow.sh` - Workflow testing
- `test-redliner-demo.sh` - Redliner demo
- `test-improved-validation.sh` - Validation testing

## Note

If you need to reference the original email-based workflow or implementation details, these documents contain that information. For current project direction, see:
- `/README.md` - Current project overview and roadmap
- `/CLAUDE.md` - AI assistant context and project details
